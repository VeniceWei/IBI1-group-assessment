# IBI1-group-assessment
这个是我们小组的共享repo，大家把这个repo克隆到自己的电脑，然后就在这个repo里面编辑，上传自己的代码，也可以看小组里其他同学写的代码

克隆:在vs code中cd到你想保存这个repo的文件夹，在终端里输入
     git clone "https://github.com/VeniceWei/IBI1-group-assessment.git"

将repo克隆到本地之后，小组成员上传到git repo里的文件是不会自动保存到你自己的本地repo里的，如果想要同步，打开终端，cd对应文件夹，在终端输入 git pull 就可以让你的本地repo和git repo同步了
